# Hash tables

## JS Object

```javascript
const a = {
  b: 1,
};
```

- only allows string keys. when you pass other datatype as key, they will be stringified
- no order

## JS Map

```javascript
const a = new Map();
```

- map allows any datatype as keys.
- maintains insertion order

## JS Set

```javascript
const a = new Set();
```

- only stores keys
